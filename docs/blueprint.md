# **App Name**: Tiny Thali

## Core Features:

- Personalized Setup: Welcome screen with input for baby's name and age to personalize the experience.
- Allergy Input: Allergy selection screen with a multi-select list of common allergens and a text input for custom allergies.
- Favorite Foods: Food preferences screen where users can select up to 3 favorite foods represented by selectable cards.
- Suggested Food Pairings: Suggest three food pairings, using information about the selected food preferences. Use food nutrition database tool, in conjunction with the selected food preferences.

## Style Guidelines:

- Background color: Soft cream background (#F8F5F0) to create a warm and inviting feel.
- Primary color: Muted blush pink (#E3A3B2) for a gentle, baby-friendly aesthetic.
- Accent color: Light mint green (#B2E3D9) as a complementary accent for buttons and interactive elements.
- Use of playful, slightly rounded fonts, focusing on readability and a friendly feel.
- Custom illustrations for each of the selectable food cards.
- Rounded input fields and cards with subtle soft shadows for a modern, friendly interface.
